# Project Summary
The project is a cloud-based inventory management system designed to optimize the management of inventory, suppliers, invoices, and shipments for businesses. It provides real-time data synchronization and insightful analytics to enhance operational efficiency and support informed decision-making. The system is accessible on various devices, promoting seamless operations and an improved user experience. The latest enhancements include a relationship protection feature that prevents the deletion of related entities, ensuring data integrity.

# Project Module Description
- **Dashboard**: Overview of key inventory metrics.
- **Items Management**: Add, edit, and delete item records.
- **Suppliers Management**: Manage supplier details, including contact and payment terms.
- **Invoices Management**: Generate and track invoices with automatic numbering.
- **Locations Management**: Monitor storage locations and their capacities.
- **Shipping Management**: Oversee shipment details, including tracking statuses and costs.
- **Reports**: Generate comprehensive inventory and supplier performance reports.
- **Real-time Updates**: Reflect changes immediately across all pages.
- **Enhanced Interface**: Improved user interface with a sidebar for easy navigation.
- **Data Mapping**: Implementation of a data mapping layer to handle discrepancies between frontend and backend data structures.
- **Relationship Protection**: New feature preventing the deletion of entities linked to others.

# File Description Inventory
- **src/App.tsx**: Main application file that sets up routing and layout.
- **src/pages/**: Contains page components for various functionalities (e.g., Dashboard, Items, Suppliers, Invoices, Locations, Shipping).
- **src/components/**: UI components used throughout the application.
- **src/data/mockData.ts**: Mock data for testing and development purposes.
- **src/types/index.ts**: TypeScript interfaces for type safety across the application, updated for consistency with database schema.
- **src/lib/autoNumber.ts**: Library for generating unique auto-incrementing numbers.
- **src/lib/localStorage.ts**: Library for managing local storage of various entities.
- **src/lib/supabase.ts**: Configuration for Supabase client and data types.
- **src/lib/supabaseStorage.ts**: Utility for managing data operations with Supabase, including the `CountersStorage` class.
- **src/lib/relationshipChecker.ts**: New library for checking entity relationships before deletion.
- **src/lib/dataMapper.ts**: Library for transforming data between camelCase (Frontend) and snake_case (Database).

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.
- **TanStack Query**: Data fetching and state management library.
- **Radix UI**: Component library for accessible UI primitives.
- **Supabase**: Backend as a service for database management.

# Usage
1. Install dependencies: `pnpm install`
2. Lint the code: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Run the application: `pnpm run dev`
