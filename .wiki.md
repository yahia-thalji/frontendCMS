# Project Summary
The project is an advanced import inventory management system that allows businesses to efficiently track items, suppliers, invoices, locations, and shipments. Its primary goal is to enhance operational efficiency and provide insightful analytics for better decision-making. The system automatically saves user inputs across all pages, eliminating the need for manual data saving. Recent updates have focused on improving error handling and data presentation, particularly addressing issues related to undefined values in the application.

# Project Module Description
- **Dashboard**: Provides an overview of key metrics and statistics related to inventory management.
- **Items Management**: Facilitates the management and tracking of items, allowing users to add, edit, and delete records.
- **Suppliers Management**: Manages supplier information, including contact details and payment terms.
- **Invoices Management**: Generates and tracks invoices with features for automatic number generation and comprehensive data handling.
- **Locations Management**: Manages storage locations and monitors their capacities.
- **Shipping Management**: Oversees shipment details, including tracking statuses and costs.
- **Reports**: Generates reports for inventory analysis, supplier performance, and cost distribution.

# File Description Inventory
- **src/App.tsx**: Main application file, sets up routing and layout.
- **src/pages/**: Contains page components for different functionalities (e.g., Dashboard, Items, Suppliers, Invoices, Locations, Shipping).
- **src/components/**: UI components used throughout the application.
- **src/data/mockData.ts**: Mock data used for testing and development.
- **src/types/index.ts**: TypeScript interfaces for type safety across the application.
- **src/lib/autoNumber.ts**: Library for generating unique auto-incrementing numbers for various entities.
- **src/lib/localStorage.ts**: Library for managing local storage of items, suppliers, locations, shipments, and invoices.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.
- **TanStack Query**: Data fetching and state management library.

# Usage
1. Install dependencies: `pnpm install`
2. Lint the code: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Run the application: `pnpm run dev`
