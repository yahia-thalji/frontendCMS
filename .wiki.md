# Project Summary
The project is a cloud-based inventory management system tailored for businesses to efficiently track and manage inventory, suppliers, invoices, locations, and shipments. By enhancing operational efficiency through real-time data synchronization, it empowers users with insightful analytics for informed decision-making. The fully integrated cloud solution allows seamless access from any device, ensuring uninterrupted operations regardless of location.

# Project Module Description
- **Dashboard**: Overview of key metrics related to inventory management.
- **Items Management**: Add, edit, and delete item records.
- **Suppliers Management**: Manage supplier details, including contact and payment terms.
- **Invoices Management**: Generate and track invoices with automatic number generation.
- **Locations Management**: Monitor storage locations and their capacities.
- **Shipping Management**: Oversee shipment details, including tracking statuses and costs.
- **Reports**: Generate comprehensive reports for inventory analysis and supplier performance.
- **Cloud Migration**: Seamlessly migrate existing local data to the cloud.

# File Description Inventory
- **src/App.tsx**: Main application file that sets up routing and layout with cloud integration.
- **src/pages/**: Contains page components for various functionalities (e.g., Dashboard, Items, Suppliers, Invoices, Locations, Shipping, Reports).
- **src/components/**: UI components used throughout the application.
- **src/data/mockData.ts**: Mock data for testing and development purposes.
- **src/types/index.ts**: TypeScript interfaces for type safety across the application.
- **src/lib/autoNumber.ts**: Library for generating unique auto-incrementing numbers.
- **src/lib/localStorage.ts**: Library for managing local storage of various entities.
- **src/lib/firebase.ts**: Firebase configuration and setup.
- **src/lib/cloudStorage.ts**: Functions for managing cloud storage operations.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.
- **TanStack Query**: Data fetching and state management library.
- **Radix UI**: Component library for accessible UI primitives.
- **Firebase**: Cloud-based database and authentication service.

# Usage
1. Install dependencies: `pnpm install`
2. Lint the code: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Run the application: `pnpm run dev`
