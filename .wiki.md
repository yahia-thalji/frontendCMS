# Project Summary
The project is a cloud-based inventory management system designed for businesses to efficiently track and manage their inventory, suppliers, invoices, locations, and shipments. It enhances operational efficiency through real-time data synchronization and offers insightful analytics for better decision-making. The system is now fully cloud-integrated, allowing users to access it from any device, ensuring seamless operations regardless of location.

# Project Module Description
- **Dashboard**: Provides an overview of key metrics and statistics related to inventory management.
- **Items Management**: Manage and track items, enabling users to add, edit, and delete records.
- **Suppliers Management**: Handle supplier information, including contact details and payment terms.
- **Invoices Management**: Generate and track invoices with automatic number generation and comprehensive data handling.
- **Locations Management**: Manage storage locations and monitor their capacities.
- **Shipping Management**: Oversee shipment details, including tracking statuses and costs.
- **Reports**: Generate reports for inventory analysis, supplier performance, and cost distribution, now utilizing real data.
- **Cloud Migration**: New feature to migrate existing local data to the cloud seamlessly.

# File Description Inventory
- **src/App.tsx**: Main application file, sets up routing and layout with cloud integration.
- **src/pages/**: Contains page components for different functionalities (e.g., Dashboard, Items, Suppliers, Invoices, Locations, Shipping, Reports).
- **src/components/**: UI components used throughout the application.
- **src/data/mockData.ts**: Mock data used for testing and development.
- **src/types/index.ts**: TypeScript interfaces for type safety across the application.
- **src/lib/autoNumber.ts**: Library for generating unique auto-incrementing numbers for various entities.
- **src/lib/localStorage.ts**: Library for managing local storage of items, suppliers, locations, shipments, and invoices.
- **src/lib/firebase.ts**: Firebase configuration and setup.
- **src/lib/cloudStorage.ts**: Functions for managing cloud storage operations.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.
- **TanStack Query**: Data fetching and state management library.
- **Radix UI**: Component library for accessible UI primitives.
- **Firebase**: Cloud-based database and authentication service.

# Usage
1. Install dependencies: `pnpm install`
2. Lint the code: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Run the application: `pnpm run dev`
