# Project Summary
The project is a comprehensive management system for import inventory that facilitates efficient tracking of items, suppliers, invoices, locations, and shipments. It is designed to streamline operations, enhance efficiency, and provide detailed analytics for informed decision-making in inventory management. The system features automatic data saving across all pages, ensuring that user inputs are retained without manual saving actions. Recent updates have improved error handling and data presentation in the dashboard.

# Project Module Description
- **Dashboard**: Overview of key metrics and statistics related to inventory management.
- **Items Management**: Manage and track items, including adding, editing, and deleting records.
- **Suppliers Management**: Manage supplier information, including contact details and payment terms.
- **Invoices Management**: Generate and track invoices with enhanced features for automatic number generation.
- **Locations Management**: Manage storage locations and track their capacities.
- **Shipping Management**: Manage shipment details, including tracking statuses and costs.
- **Reports**: Generate reports for inventory analysis, supplier performance, and cost distribution.

# File Description Inventory
- **src/App.tsx**: Main application file, sets up routing and layout.
- **src/pages/**: Contains page components for different functionalities (e.g., Dashboard, Items, Suppliers, Invoices, Locations, Shipping).
- **src/components/**: UI components used throughout the application.
- **src/data/mockData.ts**: Mock data used for testing and development.
- **src/types/index.ts**: TypeScript interfaces for type safety across the application.
- **src/lib/autoNumber.ts**: Library for generating unique auto-incrementing numbers for various entities.
- **src/lib/localStorage.ts**: Library for managing local storage of items, suppliers, locations, shipments, and invoices.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.
- **TanStack Query**: Data fetching and state management library.

# Usage
1. Install dependencies: `pnpm install`
2. Lint the code: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Run the application: `pnpm run dev`
