# خطة تطوير برنامج إدارة حاويات الاستيراد

## الملفات المطلوبة:

### 1. الصفحات الرئيسية:
- `src/pages/Dashboard.tsx` - لوحة التحكم الرئيسية
- `src/pages/Items.tsx` - إدارة الأصناف
- `src/pages/Suppliers.tsx` - إدارة الموردين
- `src/pages/Invoices.tsx` - إدارة الفواتير
- `src/pages/Locations.tsx` - إدارة مواقع التخزين
- `src/pages/Shipping.tsx` - إدارة الشحن
- `src/pages/Reports.tsx` - التقارير والتحليلات

### 2. المكونات المساعدة:
- `src/components/Sidebar.tsx` - الشريط الجانبي للتنقل
- `src/components/Header.tsx` - رأس الصفحة
- `src/components/Layout.tsx` - تخطيط الصفحة العام

### 3. الواجهات والأنواع:
- `src/types/index.ts` - تعريف الواجهات والأنواع

### 4. البيانات الوهمية:
- `src/data/mockData.ts` - بيانات تجريبية للعرض

### 5. التحديثات المطلوبة:
- تحديث `src/App.tsx` لإضافة التوجيه
- تحديث `index.html` لتغيير العنوان
- تحديث `src/pages/Index.tsx` لتوجيه إلى لوحة التحكم

## الميزات الأساسية:
1. واجهة عربية كاملة مع دعم RTL
2. إدارة الأصناف والموردين
3. نظام الفواتير مع حساب التكاليف
4. إدارة مواقع التخزين
5. تتبع الشحنات والحاويات
6. توزيع التكلفة على الأصناف
7. تقارير شاملة وتحليلات
